Taller #2: Condicionales

Enunciado 4: La video tienda que presta sus servicios de alquiler de películas a los usuarios del barrio el Porvenir, requiere de una aplicación que permita registrar el alquiler de las películas que adquieren sus usuarios. Para cada usuario se debe permitir la opción de alquilar película, consultar películas disponibles y recibir película en la video tienda con la opción de realizar anotaciones sobre estas si se llegan a presentar daños u otra novedad sobre la película.

Código de solución: 

  Proceso Condicionales_4
  Definir opcion, aux Como Entero;
  Definir pelicula1, pelicula2, pelicula3, obs Como Caracter;
  Definir estadoP1, estadoP2, estadoP3 Como Logico;
  pelicula1 <- "Titanic";pelicula2 <- "Matrix";pelicula3 <- "Pulp Fiction";
  estadoP1 <- Falso; estadoP2 <- Verdadero; estadoP3 <- Verdadero;
  obs <- ""; aux <- -1;
  
  Escribir "Menú de aplicación.";
  Escribir "1) Alquilar una película.";
  Escribir "2) Consultar películas disponibles.";
  Escribir "3) Recibir película.";
  Leer opcion;
  
  AppPeliculas(opcion, aux, estadoP1, estadoP2, estadoP3, pelicula1, pelicula2, pelicula3, obs);
  
FinProceso

SubProceso AppPeliculas(opcion, aux, estadoP1, estadoP2, estadoP3, pelicula1, pelicula2, pelicula3, obs)
  
  Segun opcion Hacer
    1:
      Escribir "Ingrese el id de la película que desea alquilar: ";
      Leer aux;
      Segun aux Hacer
        1:
          Si estadoP1 Entonces
            estadoP1 <- Falso;
            Escribir "Reclamar película en caja.";
          SiNo
            Escribir "Película no disponible.";
          FinSi
        2:
          Si estadoP2 Entonces
            estadoP2 <- Falso;
            Escribir "Reclamar película en caja.";
          SiNo
            Escribir "Película no disponible.";
          FinSi
        3:
          Si estadoP3 Entonces
            estadoP3 <- Falso;
            Escribir "Reclamar película en caja.";
          SiNo
            Escribir "Película no disponible.";
          FinSi
        De Otro Modo:
          Escribir "id de película no disponible.";
      FinSegun
      
    2:
      Escribir "Películas disponibles: ";
      Escribir "------------------------";
      Si estadoP1 Entonces
        Escribir "1) ", pelicula1;
      FinSi
      Si estadoP2 Entonces
        Escribir "2) ", pelicula2;
      FinSi
      Si estadoP3 Entonces
        Escribir "3) ", pelicula3;
      FinSi
    3:
      Escribir "Ingrese el id de la película que desea devolver: ";
      Leer aux;
      Segun aux Hacer
        1:
          Si estadoP1 == Falso Entonces
            estadoP1 <- Verdadero;
            Escribir "Porfavor ingrese una observación: ";
            Leer obs;
            Escribir "---------";
            Escribir "Película entregada con la siguiente observación: ";
            Escribir obs;
          FinSi
        2:
          Si estadoP2 == Falso Entonces
            estadoP2 <- Verdadero;
            Escribir "Porfavor ingrese una observación: ";
            Leer obs;
            Escribir "---------";
            Escribir "Película entregada con la siguiente observación: ";
            Escribir obs;
          FinSi
        3:
          Si estadoP2 == Falso Entonces
            estadoP2 <- Verdadero;
            Escribir "Porfavor ingrese una observación: ";
            Leer obs;
            Escribir "---------";
            Escribir "Película entregada con la siguiente observación: ";
            Escribir obs;
          FinSi
        De Otro Modo:
          Escribir "id de película no disponible.";
      FinSegun
      
    De Otro Modo:
      Escribir "La opción seleccionada no existe, intente de nuevo.";
  FinSegun
FinSubProceso
